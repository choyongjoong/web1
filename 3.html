JavaScript

JavaScript(JavaScript/[6]), often abbreviated as JS, is a high level, dinamic, weakly typed, prototype-based, multi- paradigm, and interpreted programming language,Alongside HTML and CSS, JavaScript is one of three core technologies of World Wide Web content production.It used to make web pages interactive and provide online programs, incuding vidio games.

The majority of websites employ it, and all mordern web browsers support it without the need for plug-ins by means of a built-in JavaScript engine.Each of many JavaScript engines represent a different implemantation of JavaScript, all based on the ECMAScript specifacation, with some engines not supporting the spectfully, and many engines supporting additional featurres beyond ECMA.
